Small stash of scripts for Homey Pro:

- device-availability-logger:
  - gets all connected devices
  - creates and continuously sets corresponding Insights tag to device availability status
  - sets corresponding Insights tag to total device count minus unavailable devices

- device-connection-logger:
  - gets all connected devices
  - creates and continuously sets corresponding Insights tag to device connection status
  - sets corresponding Insights tag to total device count minus disconnected devices
